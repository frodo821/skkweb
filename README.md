# 頭が痛くなりそうなSKK

## クエリパラメータ

`?<param-name1>=<param-value1>&<param-name1>=<param-value1>&...`の形式で指定する。

| パラメータ | 効果                                                |
| :--------- | :-------------------------------------------------- |
| gaming     | `disable`を指定すると文字のエフェクトを無効にする。 |
